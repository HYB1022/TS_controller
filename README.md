## 📝 소개

## 🧰 기능
1. 자신이 이용하는 시뮬레이터에 맞게 키 맵핑
2. 키 홀드, 특수 키 지정 지원
3. 실시간 컨트롤러 상태 확인 가능
## ❓ How to?
1. 릴리즈를 다운로드 받은 후, 실행합니다.
## 🛠️ 직접 빌드하기
1. Module_Installer.bat을 실행합니다.
2. build.bat을 실행합니다.
